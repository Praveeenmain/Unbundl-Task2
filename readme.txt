# Custom Chocolate Pack Builder

This project is a web application that allows users to build their custom chocolate packs by selecting various chocolates available in the list.
#Demo
https://intrntask2.ccbp.tech/


## Features

- Displays a list of chocolates with checkboxes for selection.
- Dynamically updates the selected chocolates list and calculates the total price based on the user's selections.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/custom-chocolate-pack.git
    ```

2. Open the `task2.html` file in your browser.

## Usage

- Select the chocolates by clicking on the checkboxes.
- The selected chocolates will appear in the "Selected Chocolates" section with the total price.
